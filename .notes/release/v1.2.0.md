# WealthSnap v1.2.0 - Release Notes

**Release Date:** January 21, 2026
**Build:** Version 1.2.0 (Major Release)

---

## ✨ What's New

### 🎨 Visual & UI Overhaul
- **Profile Screen Redesign**: Transformed into a modern interface with colored accent cards, icon-based headers, and a polished `SettingItem` component for a premium look.
- **Enhanced Custom Alerts**: Replaced system alerts with themed custom modals for a unified design across all messages (Success, Error, Confirmation).
- **Refined Transaction Options**: Replaced the native alert menu with a modern Bottom Modal in the History screen, featuring better tap targets and accessibility.

### 🤖 Gemini AI & Smart Features
- **Smart Receipt Analysis**: AI now automatically analyzes uploaded receipts to extract amount, category, and notes (marked with a ✨ icon).
- **Secure Processing**: Added a non-dismissible "Processing..." overlay during AI analysis for a stable experience.
- **API Key Guide**: Added a step-by-step "How to Get API Key" guide in Gemini Settings with direct links to Google AI Studio.
- **Usage History**: Added a "Usage" section to view Gemini API usage history.

### ⚡ UX Enhancements
- **Improved Record Workflow**: The Record menu is re-clickable, forms auto-reset on type change, and better Android back button support.
- **Dynamic Modals**: Calculator and Recurring Transaction modals now adapt better to screen sizes and content length.
- **Unified PIN Entry**: Standardized PIN screens across Onboarding and Profile Settings.

### 🐛 Key Bug Fixes
- **Recurring Transactions**: Fixed a critical bug where editing a recurring entry created duplicate rules.
- **Double Alerts**: Resolved an issue where success alerts appeared twice during PIN setup.
- **Stability**: Fixed a circular dependency in the storage layer and improved app stability.

### 🔐 Security & Onboarding
- **Privacy Updates**: Updated policies to explain "Hybrid Encryption" (sensitive data encrypted, metadata raw for performance).
- **AI Key Privacy**: Added visibility toggle (eye icon) for the AI Key input.

---

## 🛠️ Technical Improvements
- **Component Architecture**: Created reusable `SettingItem`, `TransactionOptionsModal`, and `CustomAlert` components.
- **Code Refactoring**: Cleaned up imports, resolved linting warnings, and refactored AI backend services.
- **Version Automation**: Implemented automated version bumping scripts.
