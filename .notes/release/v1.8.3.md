# WealthSnap v1.8.3 – Release Notes

**Release Date:** February 11, 2026
**Build:** Version 1.8.3 (Maintenance Release)

---

## 📈 Insights

- **Interactive Trends**: The Savings Rate Trend chart is now fully interactive!
    - **Tap-to-Inspect**: Tap any point on the chart to instantly see a detailed popup with your precise Savings Rate, Income, and Expense for that month.

---

## 🌍 Native Currency Support

- **Invest in Any Currency**: You can now record investments in their native currency (e.g., USD, EUR, JPY) even if your profile is set to a different currency (e.g., PHP).
	- **Auto-Fetch Rates**: The app automatically fetches the latest exchange rate for you using the **Frankfurter API** (https://api.frankfurter.dev).
	- **Manual Override**: You can manually adjust the exchange rate if needed.
- **Native History View**: Your investment history now displays the original native amounts (e.g., $100.00)

---

## ₿ Crypto Support

- **Crypto Investment**: Added "Crypto" as a selectable investment type.

---

## 🐛 Bug Fixes

- **Monthly Unrealized P/L**: Fixed calculation logic where sold investments were incorrectly contributing to monthly unrealized gains/losses. Now correctly caps based on current holdings.
