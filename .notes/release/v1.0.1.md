# WealthSnap v1.0.1 - Release Notes

**Release Date:** January 19, 2026  
**Build:** Version 1.0.1 (Build 2)

---

## ✨ What's New

### 🎓 Onboarding Experience
- **New Onboarding Guide** - First-time users now see an interactive guide after PIN setup
  - Explains local data storage and encryption
  - Walkthrough of key features: Backup/Restore, Transactions, Recurring, and Budgets
  - Prevents accidental back navigation during onboarding
  - Accessible anytime from Profile Settings → Help & Guide

### 🎨 UI/UX Improvements
- **Profile Screen Redesign**
  - Standardized button colors for professional appearance
  - Added "Help & Guide" card with icon and better typography
  - Cleaner, more organized layout

- **Gemini AI Settings Modal**
  - Converted inline form to bottom modal for consistency
  - Improved user experience when configuring API key
  - Better mobile-friendly interface

- **Support Developer Section**
  - Subtle "Support Developer" link in About section (not intrusive)
  - PayPal donation option via bottom modal

### 🔒 Privacy & Legal
- **Updated Terms & Privacy Policy**
  - Added Children's Privacy section (COPPA compliance)
  - Clearer data encryption and storage explanations
  - Updated to reflect app's offline-first architecture

---

## 🐛 Bug Fixes
- Fixed nested card layout issues in Profile screen

- Corrected function scoping issues in onboarding flow
