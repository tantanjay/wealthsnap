# WealthSnap v1.3.1 - Release Notes

**Release Date:** January 23, 2026  
**Build:** Version 1.3.1

---

## ✨ What's New

### 📈 Enhanced Analytics Charts
- **Time Range Filters** - You can now filter both Savings and Income charts by:
  - **6M** (Last 6 Months)
  - **1Y** (Last 1 Year)
  - **3Y** (Last 3 Years)
  - **ALL** (Entire History)

- **Smart "All Time" View**
  - Charts automatically optimize for large datasets
  - Labels are intelligently hidden to prevent clutter while keeping trends visible
  - "Income Insights" (Growth/Decline %) message updates based on current view

### 💡 Savings Trend Improvements
- **Smart Dynamic Scaling**
  - Chart Y-axis now intelligently scales to fit your specific data
  - Prevents "volatile" empty space while keeping the 0% line perfectly aligned
  - Automatically switches between "Symmetric" (for mixed positive/negative data) and "Standard" (for all-positive data) modes

- **Visual Enhancements**
  - **Split-Color Trends**: Savings line turns Green/Blue when positive and Red when negative
  - **Dynamic Labels**: "Average", "Best", and "Worst" stats are now color-coded (Green/Red) based on values
  - **Cleaner Layout**: Removed distracting shadows and improved axis readability

- **Improved UI & Education**
  - Title moved outside the card (consistent with other insights)
  - Added info icon with comprehensive educational modal
  - **Visual Learning**: Modal includes color-coded examples, sample charts, and clear explanations
  - **"Reading the Chart"** section with side-by-side visual examples showing positive vs negative trends
  - Clarifies that "8 / 5" format means "count / streak" (not division) with color-coded breakdowns
  - User-friendly explanations suitable for those without financial background

### 📊 Income Analysis Projector
- **Projections Work Everywhere** - The "Month End Projection" feature (lighter bar segment) now works seamlessly across all time filters (6M/1Y/3Y/ALL), comparing current progress against the average of the selected period.

### 💎 New Financial Insight Cards
- **10-Card Expansion** - The overview section has been expanded from 5 to **10 cards (5 pairs)** for a complete financial snapshot:
  - **Financial Runway** - Know exactly how many months you can live based on current balance vs. spending.
  - **Budget Health** - Real-time tracking of categories with active budgets.
  - **Daily Average** - Monitor your daily spending pace this month.
  - **Annualized Exp.** - Projected annual spending based on current habits.
  - **Top Category** - Instant identification of your largest expense area.
  - *Plus existing metrics: Net Cash Flow, Savings Rate, Total Income, Total Expense, and Burn Rate.*

- **Interactive Financial Education**
  - **Runway Info Modal** - Tapping the info icon (ℹ️) on the Runway card provides a clear explanation of how "Financial Safety Net" is calculated.
  - **Manual Paging System** - Swiping between cards is now manual with a sleek paging dot system (active page indicator), replacing the auto-scroll for better control and readability.

- **Smart "Item-Level" Tracking**
  - Budget Health and Top Category cards intelligently fetch individual category data automatically, ensuring accuracy without any manual UI toggling.

### ⚙️ Insights Customization
- **Personalized Dashboard** - You can now customize your Insights screen to prioritize the data that matters most to you:
  - **Cards Reordering**: Rearrange the order of the overview cards.
  - **Insights Reordering**: Change the vertical order of main sections (e.g., move "Smart Alerts" to the top).
  - **Persistent Settings**: Your layout preferences are saved automatically and persist across app restarts.
  - **Smooth Navigation**: A new settings gear icon (⚙️) provides quick access to customization modals with a seamless back-and-forth flow.

## 📝 For Testers
