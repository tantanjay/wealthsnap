# WealthSnap v1.3.1 - Release Notes

**Release Date:** January 23, 2026  
**Build:** Version 1.3.1

---

## ✨ What's New

### 📈 Enhanced Analytics Charts
- **Time Range Filters** - You can now filter both Savings and Income charts by:
  - **6M** (Last 6 Months)
  - **1Y** (Last 1 Year)
  - **3Y** (Last 3 Years)
  - **ALL** (Entire History)

- **Smart "All Time" View**
  - Charts automatically optimize for large datasets
  - Labels are intelligently hidden to prevent clutter while keeping trends visible
  - "Income Insights" (Growth/Decline %) message updates based on current view

### 💡 Savings Trend Improvements
- **Smart Dynamic Scaling**
  - Chart Y-axis now intelligently scales to fit your specific data
  - Prevents "volatile" empty space while keeping the 0% line perfectly aligned
  - Automatically switches between "Symmetric" (for mixed positive/negative data) and "Standard" (for all-positive data) modes

- **Visual Enhancements**
  - **Split-Color Trends**: Savings line turns Green/Blue when positive and Red when negative
  - **Dynamic Labels**: "Average", "Best", and "Worst" stats are now color-coded (Green/Red) based on values
  - **Cleaner Layout**: Removed distracting shadows and improved axis readability

- **Improved UI & Education**
  - Title moved outside the card (consistent with other insights)
  - Added info icon with comprehensive educational modal
  - **Visual Learning**: Modal includes color-coded examples, sample charts, and clear explanations
  - **"Reading the Chart"** section with side-by-side visual examples showing positive vs negative trends
  - Clarifies that "8 / 5" format means "count / streak" (not division) with color-coded breakdowns
  - User-friendly explanations suitable for those without financial background

### 📊 Income Analysis Projector
- **Projections Work Everywhere** - The "Month End Projection" feature (lighter bar segment) now works seamlessly across all time filters (6M/1Y/3Y/ALL), comparing current progress against the average of the selected period.

---

## 🐛 Bug Fixes
- Fixed "ugly" chart clutter when viewing long history
- Resolved visual issue where 0% grid line was not aligned with chart split
- Corrected "red line offset" in Savings Trend chart
- Fixed X-axis label overlapping on dense datasets

## 🛠️ Technical Changes
- Improved chart performance with memoized data calculations
- Refactored `SavingsRateTrend` and `IncomeAnalysis` to share smart scaling logic concepts
- Added robust error handling for empty datasets
