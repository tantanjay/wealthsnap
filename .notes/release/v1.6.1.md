# WealthSnap v1.6.1 - Release Notes

**Release Date:** January 28, 2026

**Build:** Version 1.6.1 (Critical Fixes)

## Critical Fixes

### ID Generation System (Global Fix)
- **Resolved App-Wide Crashes (Android)**: Fixed a critical incompatibility with `crypto.randomUUID` that caused the app to crash whenever a unique ID was generated.
- **Impact**: This issue affected **all key features**, including User Profile creation, adding Transactions, and creating Budgets. The ID generation system has been replaced with a robust, compatible implementation.
