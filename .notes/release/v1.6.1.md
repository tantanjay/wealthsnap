# WealthSnap v1.6.1 – Release Notes

**Release Date:** January 28, 2026  
**Build:** Version 1.6.1 (Critical Fixes)

---

## Important Notice

We sincerely apologize for the disruption caused by the **v1.6.0** release. A critical issue introduced in the previous version resulted in unexpected app crashes on certain Android devices. This update (**v1.6.1**) is a mandatory stability fix and is strongly recommended for all users.

---

## What Went Wrong

### ID Generation System (App-Wide Issue)

In **v1.6.0**, a new internal ID generation mechanism relied on `crypto.randomUUID`, which is not fully supported across all Android environments used by the app runtime. As a result, any feature that attempted to generate a new unique ID would cause the app to crash.

### Affected Features

- User Profile creation
- Adding Transactions
- Creating or modifying Budgets
- Any operation requiring a newly generated internal identifier

---

## What's Fixed in v1.6.1

### Global Stability Fix

- Fixed app-wide crashes on Android caused by incompatible UUID generation
- Removed the usage of `crypto.randomUUID`
- Replaced the ID generation system with a robust, platform-compatible implementation
- Existing user data remains safe and unaffected
